# 通过 startup.bat 启动 Tomcat
> 在这种情况下，我们需要在`setclasspath.bat`文件的起始位置设置：
> - >
>`set "JAVA_HOME=D:\jdk1.8.0_101_64bit"`。

# 将 Tomcat 注册成Windows服务时
> 我们需要在`service.bat`文件的起始位置设置：
> - >
>`set "JAVA_HOME=D:\jdk1.8.0_101_64bit"`。
